hazelnut

hazelnut is an APACHE licensed library written in Python designed to provide a simple and pythonic way to parse the /proc/meminfo file on LINUX based systems.

Installation:

From source use

    $ python setup.py install

or install from PyPi

    $ pip install hazelnut

  • Search (is case insensitive):
>>> mem.search('Swap')
['SwapCached:            0 kB\n', 'SwapTotal:      16776188 kB\n', 'SwapFree:       16776188 kB\n']
  • Get memory usage as int (is case sensitive):
>>> mem.get('Inactive(anon)')
111312
